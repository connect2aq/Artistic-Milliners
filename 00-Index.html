Object Types & Variations

The mockups show Fabric sample workflows. Similar objects exist for:

  • • Apparel Sample Development - Apparel Sample Iterations → Sample Apparel Request (SAR)
  • • Yarn Sample Development - Yarn Sample Iterations → Sample Yarn Request (SYR)
  • • All three types share the same structure: Development → Iterations → R&D Request → SDO → Sample Request

Complete Workflow Path

Account → Opportunity → CSR → Sample Development → Iterations → SDO → Sample Request → Commercial Proposal → Customer Order
