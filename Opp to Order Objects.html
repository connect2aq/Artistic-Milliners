Key Points

Customer Sample Request (CSR):

• Structured intent from discovery

• No execution commitment

• Multi-material line items possible

Sample Development:

• Cluster-specific (Apparel/Fabric/Yarn)

• Optional R&D per iteration

• SDO executes in ERP

Sample Issuance:

• Can occur with or without development

• Separate requests per cluster (SAR/SFR/SYR)

• Each has line items
